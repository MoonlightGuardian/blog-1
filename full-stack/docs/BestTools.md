### 扫描自动连接 Wifi

在线工具：[📡 WiFi Card](https://wificard.bdw.to/)

工具步骤：

1. 输入 WiFi 的网络名，即手机要链接的 WiFi 名字，如： `TP-LINK_BXXXXX`，`ChinaNet-XXXX`, `CMCC-XXXX`;
2. 输入对应 WiFi 的密码；
3. 分享二维码给家人、朋友、或同事，或者打印二维码放到公共易发现的地方；

工具优点：

1. 一次生成（劳动），永久适用（躺赢）；
2. 省掉 Network 中查找对应 WiFi名，并手动输入密码；
